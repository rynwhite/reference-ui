# Button

The `Button` component is a primitive that extends the native `<button>` element with design system styling.

## Basic usage

<Button>Default button</Button>

## Multiple variants

<div style={{ display: 'flex', gap: '0.5rem', flexWrap: 'wrap' }}>
  <Button>Primary</Button>
  <Button>Secondary</Button>
</div>
